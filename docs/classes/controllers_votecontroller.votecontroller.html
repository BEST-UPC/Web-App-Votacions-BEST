    • Defined in src/controllers/VoteController.ts:88

    If the user making the request has access to the poll, is the same as the vote's user and the poll is open it adds or updates the vote.

    throws

    Error 400 if the body is not a valid vote or missing.

    throws

    Error 401 if the user is not the same or has no access or the poll is closed or the delegation is not present.

    throws

    Error 404 if the user or poll is not found.

    Parameters

    • userId: string

      id of the user making the request.

    • body: unknown

      vote to add or update. It should be a valid IVote.

    • Optional delegatedId: string

      id of the user who we will vote for.

    Returns Promise<boolean>

    true if the vote could be added/updated or false if otherwise and no errors arised.

Static getVote

  • getVote(userId1: string, userId2: string, pollId: string): Promise<IVote>
    • Defined in src/controllers/VoteController.ts:27

    It returns the vote with the ids provided. More precisely, the vote (if it exists) will be returned if:

    • The poll is public and the user making the request has access to the poll.
    • The poll is private and the user making the request is the same as the vote's user and it has access to the poll.
    throws

    Error 401 if the user is not authorized to get that vote or poll.

    throws

    Error 404 if the user or poll or vote is not found.

    Parameters

    • userId1: string

      id of the user making the request.

    • userId2: string

      id of the vote's user to get.

    • pollId: string

      id of the poll.

    Returns Promise<IVote>

    Returns the vote requested or null.
